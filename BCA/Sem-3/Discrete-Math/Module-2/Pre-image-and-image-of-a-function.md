# Pre-image-and-image-of-a-function
A function $f: A \rightarrow B$ such that for each $a \in A$, there exist a unique $b \in B$, such that $(a,b) \in R$ then its called the **pre image** of $f$ and $b$  is  called the image of $f$ .
A functions represents a relationship between a set of inputs (domain) and their corresponding  output (range), Where each input has exactly one output.
## Domain
- The set of all possible input values for which the function is defined is called domain.
## Range
- The set of all possible output values produced by the function when the input values from the domain are plugged in.
## Co-Domain
- The co-domain of a function in the set of all possible output values that the function could potentially produce.
